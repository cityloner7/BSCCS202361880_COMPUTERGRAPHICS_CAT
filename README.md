 Project Description
This project contains architectural and engineering plans for a 14-floor commercial building designed using 
ArchiCAD 29. The building includes offices, reception,washrooms, elevators, and staircases.

Drawing Contents
(a) Architectural Floor Plan
- Walls, doors, windows, staircases and elevator shafts
- Office layout, corridors, washrooms and reception
- Correct layers and line types
- Dimensions and labels

(b) Electrical Wiring Layout
- Main power supply line
- Distribution board (DB-Main)
- Lighting points and socket outlets
- Switch positions and data ports
- Electrical symbols and legend

(C) Water Supply and Plumbing Plan
- Main water tank and pump room
- Cold water risers (CWR)
- Hot water risers (HWR)
- Stack vent (SV)
- Washroom connections
- Waste water drainage system

(d) WiFi and Network Cabling Plan
- IDF-G distribution frames
- WiFi access points (WAP-1 to WAP-17)
- CAT6 cabling routes
- Fiber optic backbone
- Wired data drops
- Signal coverage zones
